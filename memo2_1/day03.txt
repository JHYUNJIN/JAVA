상수
	항상 그대로 인 수.
	값을 변경 할 수 없도록 한다.

	final 자료형 상수명 = 값;

형변환
	- 자동 형변환 (암묵적)
		정수 + 정수 = 정수
		정수 + 실수 = 실수
		 3 + 0.0 = 3.0

	- 강제 형변환 (명시적)
		(자료형) 값
		(double) 3

문자열 형변환
	"안"+"녕" = "안녕" 
	"안녕"+1 = "안녕1"

	1. 다른 자료형을 문자열로
		문자열과 다른 일반 자료형을 연결하면 결과는 항상 문자열이 된다.
	2. 문자열을 다른 자료형으로
		일반 자료형은 일반 자료형 끼리만 형벼환이 가능하다.
		문자열 타입은 클래스타입이므로 일반 자료형의 클래스타입의 도움을 받아야한다.

		Integer.parseInt(""); --> 문자열에서 변환된 정수값
		Float.parseLoat(""); --> 문자열에서 변환된 실수값
		Double.parseDouble(""); --> 문자열에서 변환된 실수값

입력 클래스
	Scanner sc = new Scanner(SYSTEM.in);
	String name = "정현진";

입력 메소드(콘솔창에서 커서를 깜빡여주기 위해 사용)
	next() : 사용자가 입력한 문자열 값
		- 사용자가 입력한 값 중 공백 또는 줄바꿈 문자를 구분점으로 각 문자열을 분리한다.
	
	nextLine() : 사용자가 입력한 문자열 값
		- 공백 또는 줄바꿈 문자도 값으로 취급하기 때문에 그대로 입력 받는다.
