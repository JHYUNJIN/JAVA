파일 다루기

File클래스
파일 또는 디렉토리를 다루는 클래스
입출력에 관한 작업
	File 참조변수 = new File(파일경로 or 파일이름);

**스트림(stream) //데이터를 가공하고 처리하는 인터페이스
-자바는 파일이나 콘솔의 입출력을 직접다루지않고 스트림이라는 흐름을 통해 다룸
-운영체제에 의해 생성되는 가상의 연결고리를 의미하고 중간 매개자 역할을 한다.

	JAVA 프로그램 <--------> OS(운영체제) <------------> 디스크(파일,디렉토리)
					모니터, 프린터, 네트워크,...
**절대경로
	물리적인 경로
	ex) D:\jhj\java\day09

**상대경로			
	현재 동작하고 있는 파일을 중심으로 상대적인 경로
	ex) day09.txt, 디렉토리명/day09.txt,.../day08.txt							
exits() : 파일이 실제 존재하는지 여부(boolean)
isDirectory() : 해당 경로가 디렉토리인지 여부(boolean)
length() : 파일의 데이터길이를 반환(byte), 한글은 3byte, 영어/특수문자/공백 1byte
mkdir() : 디렉토리를 생성
createNewFile() : 파일을 생성
